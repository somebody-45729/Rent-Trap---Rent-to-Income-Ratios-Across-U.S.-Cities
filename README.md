# Rent Trap: Tracking Rent-to-Income Ratios Across U.S. Cities
As rent rises higher than current wages, what cities have become unafforable and when?

Anchor: 30% rule - Guideline that rent should not exceed 30% of gross income.
When median rent crosses that threshold of median income, it is considered "cost-burdened' territory.
